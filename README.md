# Simulador Insertion Sort

Proyecto desarrollado en Python y Streamlit.

## Funciones

- Ingreso manual de datos.
- Generación aleatoria.
- Animación visual.
- Medición de tiempo.
- Conteo de iteraciones.

## Algoritmo

Insertion Sort ordena los elementos insertándolos uno a uno en su posición correcta dentro de la parte ya ordenada del arreglo.
